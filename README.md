# Google homepage
Google homepage est un projet de la formation "projet odin" consistant à recréer la page d'accueuil du moteur de recherche Google.

## Compétences
[ ] Déplacer une div sur une page
[ ] Ancrer une div en haut et en bas de la page
[ ] Identifier la couleur de l'arrière plan d'une page
[ ] Récupérer l'URL d'une image
[ ] Centrer un élément horizontalement
[ ] Mise en place d'un CSS
[ ] Construction d'un formulaire basique
